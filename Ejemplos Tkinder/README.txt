
Tkinter - Scripts básicos (Widgets, Layouts y Eventos)
=====================================================

Contenido:
  01_widgets_basicos.py    -> Demostración de los widgets fundamentales de Tkinter.
  02_layout_pack.py        -> Ejemplo del administrador de geometría pack().
  03_layout_grid.py        -> Ejemplo del administrador de geometría grid().
  04_layout_place.py       -> Ejemplo del administrador de geometría place().
  05_eventos_basicos.py    -> Manejo de eventos: teclado, mouse, ventana y 'command'.
  06_ejemplo_completo.py   -> App de muestra combinando widgets, layouts y eventos.

Requisitos:
  - Python 3.x con Tkinter (suele venir incluido).
  - Ejecutar en Windows, macOS o Linux con entorno gráfico.

Ejecución:
  - Abra una terminal/cmd en esta carpeta y ejecute:
      python 01_widgets_basicos.py
      python 02_layout_pack.py
      python 03_layout_grid.py
      python 04_layout_place.py
      python 05_eventos_basicos.py
      python 06_ejemplo_completo.py
